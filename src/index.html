Brand Guide

Introduction

Kono is Hawaiian and means "invite" - "you are invited" and thats exactly what the design is supposed to reflect. The color and form language should be appealing and simple. But nevertheless it should radiate summer feelings and merriment. Due to this reason the design focuses on a simple and clear form language and the colors are used as a highlight.

CSS Methodology

Code Style Rules

Kono uses Codeguide.co as a base for Style Rules

Line Length

maximum line length: 90 characters

Every Declaration on a new line

Closing braces } of declaration blocks on a new line.

Indentation

Use two spaces
No tabs or mix of tabs and spaces (except line break inside parentheses)

Spacing

Space around operators : * + - = > <

One SPACE before { for declaration

Hexadecimal Notations

Only 3 character where possible

Lowercase hex values #fff

Valid CSS

Use of valid CSS code where possible

Comments

comments for non self-explainable code

Units

em as standard unit padding: 2em;

Avoid the specifying units for zero values padding: 0;

Naming Conventions

Kono follows the BEM naming conventions
Ensuring to have a cosistent names,

  • short as posssible, long as neccessary
