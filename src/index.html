Brand Guide

Introduction

Kono ist hawaiianisch für "einladen" - "du bist eingeladen" und genau das soll sich auch im Design wiederspiegeln. Die Farb- und Formsprache soll einladend und simple sein, aber trotzdem sommerlich und fröhlich wirken. Aus diesem Grund wird auf eine klare und simple Formsprache gesetzt und mit Hilfe von Farbe Highlights gesetzt.

Colors

Die obere Pallette stellt die Hauptfarben dar, die an den frischen Farben einer Ananas angelehnt sind. Die untere Farbpallette sind die passenden Border-farben, welche bei Icons und Buttons zum Einsatz kommen.

CSS Methodology

Code Style Rules

  • Use of 3 character hexadecimal notation where possible
  • Use of valid CSS code where possible
  • Indentation: Two spaces
  • comments for non self-explainable code
  • No usage of ID-Selectors

Naming Conventions

Kono follows the BEM naming conventions
Ensuring to have a cosistent names,

  • short as posssible, long as neccessary
